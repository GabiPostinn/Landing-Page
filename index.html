✅ 1. Declararea variabilelor:

JavaScript are 3 tipuri de variabile:

let nume = "Alex"; // variabilă care poate fi schimbată

const pi = 3.14; // constantă (nu se poate schimba)

var vechi = 5; // vechi, se evită în cod modern

✅ 2. Tipuri de date:

JavaScript are 7 tipuri de date de bază:

String – șir de caractere: "Salut"

Number – număr: 10, 3.14

Boolean – adevărat / fals: true, false

Array – listă: [1, 2, 3]

Object – obiect: { nume: "Alex", varsta: 18 }

null – valoare nulă

undefined – valoare nedefinită
